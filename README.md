# SECAR - Self-Evolving Context-Aware Rule Engine

🧠 **Patent-Pending AI System that learns from your behavior**

## 🌟 Features
- **Self-Evolving Rules** - AI learns your patterns and adjusts automatically
- **Context Awareness** - Adapts based on time, location, and day type
- **Before/After Learning** - See how your habits evolve (7:00 AM → 6:00 PM)
- **Confidence Levels** - High/Medium/Learning indicators
- **"Why?" Explanations** - Click any habit to see AI reasoning
- **Achievements** - Unlock badges as you progress
- **Analytics** - Track your learning journey

## 🚀 Live Demo
[https://danush88.github.io/secar-project/](https://danush88.github.io/secar-project/)

## 📋 How to Use
1. Create habits with initial times
2. Click ✓ when you complete them
3. Watch the AI learn your patterns
4. See optimized times appear automatically
